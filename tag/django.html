周一 23 十一月 2015

没人访问的博客也不是不能升级。不过最近有点懒，趁着睡不着搞了下uwsgi。 额。平台ubuntu, 你最好有个django项目，假设已经用了virtualenv，假设已经安装好nginx并启动服务。

1）首先安装uwsgi: 

pip install uwsgi

2）然后启动， 指定端口号和进程号：

uwsgi -s 127.0 …
